About Project

Admin Abuse is an independent testing client designed for any system that can run Minecraft effectively. This is a beta phase client focused on servers with little to no anti-cheat, such as play.6b6t.org, 6b8t.eagler.host, and 2b2tpvp.org. This is not a main client, but a testing ground for custom modules like New Chunks and aggressive Auto Crystal settings.

How to Build

To compile the latest development builds from source, use the following commands:

git clone https://github.com/overyr8-sys/Admin-Abuse-Client.git
cd Admin-Abuse-Client
./gradlew build

The compiled jar file will be found in build/libs/.
